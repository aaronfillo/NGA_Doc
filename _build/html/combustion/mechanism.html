mechanism.f90¶

Chemical mechanism used in NGA.

Module mechanism

subroutine mechanism/COMPTHERMODATA(H, CP, T)¶

Computes the thermodynamic data (enthalpy and specific heat) for the species at a given temperature. The returned values for H and CP are fit as functions of T.

Parameters:
  • H :: Array of species enthalpy.
  • CP :: Array of species specific heats.
  • T :: Temperature of mixture.
Called from:

finitechem_diffusivity()
subroutine mechanism/GETNSPECIES(NSPECIES)¶

Returns the number of species in the mechansim.

Parameters:NSPECIES :: Number of species in the mechanism.
subroutine mechanism/GETSPECIESNAMES(NAMES)¶

Returns the names of the species in an array of strings.

Parameters:NAMES :: Array of species names.
subroutine mechanism/GETMOLARMASS(MM)¶

Returns the molar mass of each species, in kg/mol.

Parameters:MM :: Array of molar masses.
subroutine mechanism/GETMUCOEFF(MUCOEFF)¶

Returns the species muCoeffs in kg/m.s, which is used in calculations for computing the species viscosities.

Parameters:MUCOEFF :: Array of species muCoeffs.
subroutine mechanism/GETKOVEREPS(KOVEREPS)¶

Returns the species \(k/\epsilon\), which is used in fits of the collision integrals. Here, \(k\) is Boltzmann’s constant and \(\epsilon\) is the potential well depth.

Parameters:KOVEREPS :: Array of \(k/\epsilon\) values
subroutine mechanism/GETNREACTIONS(NREACTIONS)¶

Returns the number of reactions in the mechanism.

Parameters:NREACTIONS :: Number of reactions in the mechanism.
subroutine mechanism/GETLINDRATECOEFF(TEMP, PRESSURE, K0, KINF, FC, CONCIN)¶

Returns the rate constant for Lindemann reactions. TEEMO.

subroutine mechanism/PRODRATES(C, TEMP, PRESSURE, CDOT_p, CDOT_n, W, K, M)¶

Returns the production rates of species in mole/m^3.s. TEEMO.

subroutine mechanism/COMPSTEADYSTATES(K, C, M, TEMP, PRESSURE)¶

Computes the steady state concentrations from the concentrations of computed species and rate coefficients. Concentrations of computed species may not be altered. TEEMO.
